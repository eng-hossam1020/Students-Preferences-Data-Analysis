# 🎓 Students’ Preferences Data Analysis Project
## 📌 Context

Educational institutions continuously seek to understand students’ academic preferences to improve specialization planning, student guidance, and registration processes. This project analyzes students’ academic specialization preferences across different schools using interactive data analysis and dashboarding techniques.

<b>The analysis focuses on identifying:</b>

- Student response and registration rates
- Remaining students who have not completed registration
- Most preferred academic specializations
- Preference ranking trends across schools
- Distribution percentages of academic specializations

This project was designed to support data-driven decision-making for school management and academic planning.

## 📊 Project Overview

This project provides a comprehensive analysis of students’ academic specialization preferences collected from multiple schools. The dataset includes students’ ranked preferences from first choice to fifth choice along with school information.

Using Excel Pivot Tables, Pivot Charts, Slicers, and Dashboard Visualization techniques, the project transforms raw student response data into meaningful insights and KPIs.

The dashboard enables dynamic filtering by school and specialization to monitor registration performance and preference trends interactively.

## 🎯 Project Objectives
- Analyze students’ academic specialization preferences
- Monitor student response and registration status
- Identify remaining students who have not registered
- Compare specialization popularity across schools
- Track preference ranking distribution
- Support academic planning and resource allocation

## 🛠️ Data Processing and Calculations
<b>Data Cleaning</b>

<b>The dataset was processed to ensure:</b>

- Removal of duplicate responses
- Handling blank values
- Standardization of specialization names
- Validation of registration data
  
 <b>KPI Calculations</b>
<b>Student Response</b>

<b>Calculated using:</b>

- Total number of submitted student responses
Registration Count

<b>Calculated by:</b>

- Counting registered students per school and specialization
Remaining Students

<b>Calculated using:</b>

- Remaining Students = Total Students - Registered Students

<b>Preference Analysis</b>

<b>Students’ ranked preferences were analyzed by counting:</b>

<b>Number of students selecting each specialization as:</b>
- First preference
- Second preference
- Third preference
- Fourth preference
- Fifth preference
  
<b>Percentage Calculation</b>

<b>Percentage of each academic specialization calculated using:</b>

- Specialization Percentage =
(Number of Students in Specialization ÷ Total Students) × 100
